Why Calculus?

The short answer

Calculus is the language for describing change, accumulation, and optimization.

Motion

How does position change with time?

Fields

How do quantities vary from point to point?

Learning

How do we improve a model by changing its parameters?

Motivation: functions are the objects that change; calculus measures and controls that change.

Navier-Stokes: Fluid Motion

Fluid mechanics

\[ \rho\left(\frac{\partial \mathbf u}{\partial t} +(\mathbf u\cdot\nabla)\mathbf u\right) =-\nabla p+\mu\Delta\mathbf u+\mathbf f \]

The velocity field \(\mathbf u(x,y,z,t)\) changes in time and space. Calculus measures acceleration, pressure gradients, and diffusion of momentum.

Background: fluid simulation video, YouTube.

Maxwell: Fields And Waves

Electromagnetism

\[ \nabla\cdot\mathbf E=\frac{\rho}{\varepsilon_0}, \qquad \nabla\times\mathbf B-\frac1{c^2}\frac{\partial\mathbf E}{\partial t} =\mu_0\mathbf J \]

Electric and magnetic fields are functions on space and time. The symbols \(\nabla\) and \(\frac{\partial}{\partial t}\) describe how those fields vary.

Background: Maxwell equation video, YouTube.

Einstein: Space-Time Curvature

Relativity

\[ G_{\mu\nu}+\Lambda g_{\mu\nu} =\frac{8\pi G}{c^4}T_{\mu\nu} \]

Geometry tells matter how to move; matter tells geometry how to curve. Calculus enters through curvature, rates of change, and differential geometry.

Background: BH Accretion Disk Sim 360 Continuous, NASA Goddard Space Flight Center/Jeremy Schnittman, public domain.

Schrodinger: Quantum Waves

Quantum mechanics

\[ i\hbar\frac{\partial\psi}{\partial t} =\widehat H\psi \]

The wave function \(\psi\) evolves according to a differential equation. Here calculus describes how probability waves move, spread, and interfere.

Background: quantum simulation video, YouTube.

AI: Learning By Gradients

Optimization and backpropagation

\[ L(\theta)=\frac1N\sum_{i=1}^N \ell\big(f_\theta(x_i),y_i\big), \qquad \theta_{k+1}=\theta_k-\eta\nabla_\theta L(\theta_k) \]

Backpropagation is repeated use of the chain rule to compute gradients efficiently. Modern AI learns by following those gradients.

Background: neural-network video, YouTube.

Function, Domain, Range

Definition

A function \(f\) assigns to each input \(x\) in a set \(X\) exactly one output \(f(x)\).

\[ f:X\to E, \qquad y=f(x) \]

Domain

\[ D(f)=\{x\in \mathbb{R}: f(x)\text{ is defined}\}. \]

Range

\[ E(f)=\{f(x):x\in D(f)\}. \]

Theory: Active Prelude 1.2; OpenStax 1.1; Stewart 1.1.

Graph Of A Function

Graph

The graph of \(f\) is the set of all points

\[ (x,f(x)), \qquad x\in D(f). \]

Theory: Active Prelude 1.2; OpenStax 1.1; Stewart 1.1.

Example: Domain And Range

Problem

Find the domain and range of

\[f(x)=\sqrt{x+2}.\]

Solution

For \(f(x)\) to be defined,

\[ x+2\geq 0 \Longleftrightarrow x\geq -2. \]

Therefore

\[ D(f)=[-2,\infty), \qquad E(f)=[0,\infty). \]

Practice: Active Prelude 1.2; OpenStax 1.1; Stewart 1.1.

Piecewise Defined Functions

Definition

A piecewise function uses different formulas on different parts of its domain.

Example

\[ f(x)= \begin{cases} 1-x, & x\leq -1,\\ x^2, & x>-1. \end{cases} \]

Quick checks:

\[ f(-2)=3,\qquad f(-1)=2,\qquad f(2)=4. \]

Theory: Active Prelude 1.2; OpenStax 1.1; Stewart 1.1.

Composite Functions

Definition

If \(y=f(u)\) and \(u=g(x)\), then substitution gives

\[y=f(g(x)).\]

The composite function is

\[ (f\circ g)(x)=f(g(x)). \]

Theory: Active Prelude 1.6; OpenStax 1.1; Stewart 1.3.

Example: Composition

Problem

Let

\[ f(x)=x^2, \qquad g(x)=x^3-7. \]

Find \(f\circ g\) and \(g\circ f\).

First order

\[ (f\circ g)(x)=f(g(x))=(x^3-7)^2. \]

Reverse order

\[ (g\circ f)(x)=g(f(x))=x^6-7. \]

Practice: Active Prelude 1.6; OpenStax 1.1; Stewart 1.3.

One-To-One Functions

Definition

A function \(f\) is one-to-one if it never takes the same value twice:

\[ x_1\neq x_2 \Longrightarrow f(x_1)\neq f(x_2). \]

One-to-one

\(f(x)=x^3\) is one-to-one.

Not one-to-one

\(g(x)=x^2\) is not one-to-one because

\[g(1)=1=g(-1).\]

Theory: Active Prelude 1.7; OpenStax 1.4; Stewart 1.5.

Inverse Functions

Definition

Let \(f\) be one-to-one with domain \(D\) and range \(E\).

The inverse function \(f^{-1}\) has domain \(E\) and range \(D\), and

\[ f^{-1}(y)=x \Longleftrightarrow f(x)=y. \]

Recipe
  1. Write \(y=f(x)\).
  2. Solve for \(x\) in terms of \(y\).
  3. Interchange \(x\) and \(y\).
Theory: Active Prelude 1.7; OpenStax 1.4; Stewart 1.5.

Example: Inverse Function

Problem

Find the inverse of

\[f(x)=x^3+2.\]

Solution

\[ y=x^3+2 \]

\[ x^3=y-2 \]

\[ x=(y-2)^{1/3} \]

After interchanging \(x\) and \(y\):

\[ f^{-1}(x)=(x-2)^{1/3}. \]

Practice: Active Prelude 1.7; OpenStax 1.4; Stewart 1.5.

Periodic Functions

Definition

A function \(f\) is periodic with period \(T>0\) if

\[ f(x+T)=f(x-T)=f(x) \]

whenever the expressions are defined.

Example

\(f(x)=\sin x\) has period \(2\pi\) because

\[ \sin(x+2\pi)=\sin(x-2\pi)=\sin x. \]

Theory: OpenStax 1.3; Stewart 1.3.

Increasing And Decreasing

Increasing

\[ x_1<x_2 \Longrightarrow f(x_1)<f(x_2). \]

Decreasing

\[ x_1<x_2 \Longrightarrow f(x_1)>f(x_2). \]

Examples
  • \(f(x)=x\) is increasing on \(\mathbb{R}\).
  • \(g(x)=x^2\) is decreasing on \((-\infty,0)\) and increasing on \((0,\infty)\).
  • \(h(x)=c\) is neither increasing nor decreasing under the strict definition.
Theory: Active Prelude 1.2; OpenStax 1.1; Stewart 1.1.

Symmetry

Even

\[f(-x)=f(x).\]

The graph is symmetric about the \(y\)-axis.

Odd

\[f(-x)=-f(x).\]

The graph is symmetric about the origin.

Examples

\(f(x)=x\) is odd. \(g(x)=x^2\) is even.

Theory: OpenStax 1.1; OpenStax 1.2; Stewart 1.1.

Boundedness

Definitions
  • bounded above: \(f(x)\leq M\)
  • bounded below: \(f(x)\geq m\)
  • bounded: \(|f(x)|\leq C\)
  • unbounded: no single \(C\) works
Examples

\[ f(x)=\frac{1}{1+x^2} \]

is bounded on \(\mathbb{R}\) because \(0<f(x)\leq 1\).

\[ g(x)=\frac{1}{x} \]

is bounded on \([1,\infty)\) but unbounded on \((0,\infty)\).

Why \(1/x\) is unbounded near \(0\)

For any \(C>0\), choose \(x_0=1/(C+1)\). Then \(x_0\in(0,\infty)\) and \(|g(x_0)|=C+1>C\).

Theory: OpenStax 1.1; OpenStax 1.2; Stewart 1.1.

Linear Functions

Definition

A linear function has the form

\[y=mx+b.\]

\(m\) is the slope and \(b\) is the \(y\)-intercept.

Slope

For two points \((x_1,y_1)\) and \((x_2,y_2)\) with \(x_1\neq x_2\),

\[ m=\frac{y_2-y_1}{x_2-x_1}. \]

Theory: Active Prelude 1.4; OpenStax 1.2; Stewart 1.2.

Example: Line Through Two Points

Problem

Find the slope and equation of the line through \((4,3)\) and \((2,5)\).

Solution

\[ m=\frac{5-3}{2-4}=-1. \]

Using point \((4,3)\):

\[ y-3=-1(x-4) \]

so

\[ y=-x+7. \]

Practice: Active Prelude 1.4; OpenStax 1.2; Stewart 1.2.

Polynomials

Definition

\[ P(x)=a_nx^n+a_{n-1}x^{n-1}+\cdots+a_1x+a_0. \]

If \(a_n\neq 0\), the degree is \(n\).

Degree 1

\(P(x)=mx+b\)

Degree 2

\(P(x)=ax^2+bx+c\)

Degree 3

\(P(x)=ax^3+bx^2+cx+d\)

Theory: Active Prelude 1.5; OpenStax 1.2; Stewart 1.2.

Power Functions

Definition

A power function has the form

\[ y=x^\alpha, \qquad \alpha\in\mathbb{R}. \]

The domain, range, monotonicity, and symmetry depend on \(\alpha\).

Theory: OpenStax 1.2; Stewart 1.2.

Example: Square Function

\(y=x^2\)
  • Domain: \(\mathbb{R}\)
  • Range: \([0,\infty)\)
  • Decreasing on \((-\infty,0)\)
  • Increasing on \((0,\infty)\)
  • Even

Practice: Active Prelude 1.5; OpenStax 1.2; Stewart 1.2.

Example: Reciprocal Function

\(y=1/x\)
  • Domain: \(\mathbb{R}\setminus\{0\}\)
  • Range: \((-\infty,0)\cup(0,\infty)\)
  • Decreasing on \((-\infty,0)\) and \((0,\infty)\)
  • Odd

Practice: OpenStax 1.2; Stewart 1.2.

Example: Square Root Function

\(y=\sqrt{x}\)
  • Domain: \([0,\infty)\)
  • Range: \([0,\infty)\)
  • Increasing on \((0,\infty)\)
  • No even/odd symmetry

Practice: OpenStax 1.2; Stewart 1.2.

Exponential Functions

Definition

\[ y=a^x,\qquad a>0,\quad a\neq 1. \]

Laws

\[a^x a^y=a^{x+y},\qquad \frac{a^x}{a^y}=a^{x-y}\] \[\left(a^x\right)^y=a^{xy},\qquad a^{-x}=\frac{1}{a^x}\]

More laws

\[a^x b^x=(ab)^x,\qquad \frac{a^x}{b^x}=\left(\frac{a}{b}\right)^x\]

Theory: OpenStax 1.5; Stewart 1.4.

Examples: Exponential Graphs

\(a>1\)
  • Domain: \(\mathbb{R}\)
  • Range: \((0,\infty)\)
  • Increasing on \(\mathbb{R}\)
  • Passes through \((0,1)\)

\(0 < a < 1\)
  • Domain: \(\mathbb{R}\)
  • Range: \((0,\infty)\)
  • Decreasing on \(\mathbb{R}\)
  • Passes through \((0,1)\)

Practice: OpenStax 1.5; Stewart 1.4.

Logarithmic Functions

Definition

\[ y=\log_a x,\qquad a>0,\quad a\neq 1. \]

It is the inverse of \(y=a^x\):

\[ y=\log_a x \Longleftrightarrow x=a^y. \]

Domain and range

\[ D=(0,\infty), \qquad E=\mathbb{R}. \]

Theory: OpenStax 1.5; Stewart 1.5.

Logarithm Laws

Products and quotients

\[ \log_a(xy)=\log_a x+\log_a y \]

\[ \log_a\left(\frac{x}{y}\right)=\log_a x-\log_a y \]

Reciprocals and powers

\[ \log_a\left(\frac1x\right)=-\log_a x \]

\[ \log_{a^\beta}x^\alpha=\frac{\alpha}{\beta}\log_a x \]

Theory: OpenStax 1.5; Stewart 1.5.

Examples: Logarithmic Graphs

\(a>1\)
  • Increasing on \((0,\infty)\)
  • Reflection of \(y=a^x\) across \(y=x\)

\(0 < a < 1\)
  • Decreasing on \((0,\infty)\)
  • Reflection of \(y=a^x\) across \(y=x\)

Practice: OpenStax 1.5; Stewart 1.5.

Basic Trigonometric Formulas

Sine

\[\sin^2x+\cos^2x=1\] \[\sin 2x=2\sin x\cos x\] \[\sin 3x=3\sin x-4\sin^3x\] \[\sin^2x=\frac{1-\cos 2x}{2}\]

Cosine

\[\cos 2x=\cos^2x-\sin^2x\] \[\cos 2x=2\cos^2x-1=1-2\sin^2x\] \[\cos^2x=\frac{1+\cos 2x}{2}\]

Theory: OpenStax 1.3; Stewart 1.3.

Inverse Trigonometric Functions

Arcsine

\[ y=\arcsin x \Longleftrightarrow x=\sin y \]

\[ -1\leq x\leq 1,\quad -\frac{\pi}{2}\leq y\leq\frac{\pi}{2} \]

Arccosine

\[ y=\arccos x \Longleftrightarrow x=\cos y \]

\[ -1\leq x\leq 1,\quad 0\leq y\leq\pi \]

Arctangent

\[ y=\arctan x \Longleftrightarrow x=\tan y \]

\[ x\in\mathbb{R},\quad -\frac{\pi}{2}<y<\frac{\pi}{2} \]

Theory: OpenStax 1.4; Stewart 1.5.

Hyperbolic Functions

Definitions

Hyperbolic functions are built from exponentials:

\[ \sinh x=\frac{e^x-e^{-x}}{2},\qquad \cosh x=\frac{e^x+e^{-x}}{2}. \]

Contrast

They are not periodic like sine and cosine, but they satisfy identities that look similar to trigonometric identities.

Theory: OpenStax 1.5 for exponentials; Stewart 3.11.

Examples: Hyperbolic Sine And Cosine

\(y=\sinh x\)
  • Domain: \(\mathbb{R}\)
  • Range: \(\mathbb{R}\)
  • Increasing on \(\mathbb{R}\)
  • Odd

\(y=\cosh x\)
  • Domain: \(\mathbb{R}\)
  • Range: \([1,\infty)\)
  • Decreasing on \((-\infty,0)\)
  • Increasing on \((0,\infty)\)
  • Even

Practice: OpenStax 1.5 for exponential structure; Stewart 3.11.

Hyperbolic Tangent And Cotangent

Definitions

\[ \tanh x=\frac{\sinh x}{\cosh x} \]

\[ \coth x=\frac{\cosh x}{\sinh x} \]

Domains
  • \(\tanh x\) is defined for all real \(x\).
  • \(\coth x\) is undefined at \(x=0\).
Theory: OpenStax 1.5 for exponentials; Stewart 3.11.

Hyperbolic Identities

Parity and fundamental identity

\[\sinh(-x)=-\sinh x\] \[\cosh(-x)=\cosh x\] \[\cosh^2x-\sinh^2x=1\]

Addition formulas

\[\cosh^2x+\sinh^2x=\cosh 2x\] \[\sinh(x+y)=\sinh x\cosh y+\cosh x\sinh y\] \[\cosh(x+y)=\cosh x\cosh y+\sinh x\sinh y\]

Theory: Stewart 3.11; compare identities with OpenStax 1.3.

New Functions From Old Functions

Shifts

For \(c>0\):

  • \(f(x)+c\): up
  • \(f(x)-c\): down
  • \(f(x+c)\): left
  • \(f(x-c)\): right
Stretches and reflections

For \(c>1\):

  • \(cf(x)\): vertical stretch
  • \(\frac1c f(x)\): vertical shrink
  • \(f(cx)\): horizontal shrink
  • \(f(\frac1c x)\): horizontal stretch
  • \(-f(x)\) and \(f(-x)\): reflections
Theory: Active Prelude 1.8; OpenStax 1.1; Stewart 1.3.
